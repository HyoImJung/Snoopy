# STAMPEDE
콘솔 환경에서 구현한 턴제 전략 게임. 기사, 궁수, 사제, 마법사 4종의 아군 유닛으로 웨이브별로 몰려오는 몬스터들로부터 타워를 방어하는 것이 목표입니다.

붉은 달이 뜬 밤, 대륙 전역의 몬스터들이 '시원의 마석'을 차지하기 위해 왕국으로 일제히 질주(Stampede)합니다. 국왕의 소집령에 응한 4명의 영웅이 되어 성문 앞을 막아서고, 4번의 웨이브 끝에 등장하는 드래곤까지 격파하세요.

## 실행 방법
1. 저장소를 클론하거나 ZIP으로 내려받습니다.
   ```bash
   git clone https://github.com/<your-repo>/STAMPEDE.git
   ```
2. Visual Studio 2022로 솔루션 파일(`.sln`)을 엽니다.
3. 상단 메뉴 **빌드 → 일괄 빌드(Batch Build)** 를 엽니다.
4. 목록에서 **Release 구성**(`Release`, `x64` 권장)에 체크한 뒤 **빌드** 버튼을 눌러 빌드합니다.
5. 빌드가 완료되면 생성된 출력 폴더의 **`Release`(또는 `x64/Release`) 폴더로 이동**합니다.
6. 폴더 안의 **`.exe` 파일을 더블클릭**하여 게임을 실행합니다.

> **사운드 파일 배치**: BGM·효과음(`.wav`)은 `.exe`와 **같은 폴더(`Release` 폴더)** 에 있어야 재생됩니다. (예: `Prologue.wav`, `Loby.wav`(로비·튜토리얼 BGM), `Wave_1.wav` ~ `Wave_4.wav`, `Game_victory.wav`, `Game_over.wav`, `Submit.wav` 등을 `Release` 폴더에 복사)

> **한글 표시**: 한글 출력을 위해 **Windows Terminal** 사용을 권장합니다. 일반 콘솔(conhost)에서 한글이 깨질 경우 콘솔 속성에서 글꼴을 **굴림체**로 변경하면 정상 출력됩니다.

## 게임 흐름
1. **프롤로그** — 게임의 배경 스토리를 페이지 단위로 감상합니다. (`Enter`: 다음, `Backspace`: 건너뛰기)
2. **튜토리얼 선택** — 프롤로그 후 튜토리얼 진행 여부를 선택합니다. 튜토리얼에서는 기사 1명으로 **이동 → 대각선 이동 → 보물상자 개봉 → 적 접근 → 공격 → 턴 종료 → 스킬 사용**의 7단계를 익히고 마지막에 오크왕을 처치합니다.
3. **본 게임 (웨이브 방어)** — 총 4개의 웨이브를 순서대로 막아냅니다. 모든 적을 처치하면 다음 웨이브로 진행하며, 4웨이브의 드래곤을 처치하면 **승리**합니다.
4. **패배 조건** — 타워(기지)의 HP가 0이 되면 패배합니다.

## 핵심 시스템
- **AP(행동력) 시스템**: 이동·스킬 사용에 AP가 소모되며, 턴이 끝날 때마다 일정량 회복됩니다.
- **턴제 전투**: 플레이어 턴에 모든 유닛을 조작한 뒤(`z` 또는 전원 행동 완료) 적 AI 턴이 진행됩니다.
- **이동/공격 범위 표시**: 선택한 유닛의 이동 가능 칸(`*`)과 공격 가능 칸(빨간 `*`)이 맵에 표시됩니다. (궁수가 신속 이동을 모두 쓰면 이동 표시는 사라지고 공격 표시만 남습니다.)
- **데미지 표시**: 피해를 주거나 받을 때 해당 칸에 빨간 숫자로 **실제 적용된 피해량**이 잠시 표시됩니다. (방어력·약점·연속타격 등 모두 반영)
- **상태/스탯 표시**: 보호막(청록 배경)·감전(노란 배경)·기절(회색 배경)·매혹(보라 배경) 등 상태가 맵 타일과 우측 패널(`(기절)`/`(매혹)`/`(감전)` 등)에 함께 표시되고, 우측 패널에는 각 유닛의 체력·공격력·**방어력**·사거리도 표시됩니다.
- **보물상자**: 웨이브마다 1개가 맵에 등장하며, 인접해서 `e`(AP 20 소비)로 열면 무작위 **영구 버프**를 얻습니다.
- **절차적 지형**: 웨이브가 진행될수록 산·호수 같은 장애물이 점점 많아지며, 진입로는 항상 확보됩니다.
- **세이브 / 로드**: 슬롯 단위로 진행 상황(유닛 스탯·보물상자·지형 포함)을 저장하고 불러올 수 있습니다.
- **사운드**: 프롤로그·웨이브별 BGM, 승리/패배 음악, 효과음을 지원합니다.

### 유닛별 스킬
**기사 (Knight)**
- **회전베기** (AP 15) — 주변(대각선 포함) 이동 범위 내 모든 적에게 피해
- **투구깨기** (AP 20) — 인접한 적 1명에게 2배 피해 + 기절
- **도발** (AP 10) — 모든 적이 기사를 집중 공격하도록 유도

**궁수 (Archer)**
- **관통샷** (AP 15) — 직선 방향 최대 2명의 적을 관통 공격
- **신속 이동** (AP 1) — 최대 3칸 이동, 이동을 마치면 제자리에서 공격만 가능
- **삼중 타격** (AP 20) — 다음 공격을 3회 연속 타격으로 강화

**사제 (Priest)**
- **신의 축복** (AP 15) — 맵의 모든 아군 HP 15 회복
- **정화** (AP 20) — 직선 방향 적에게 피해, 언데드는 즉사
- **신의 가호** (AP 15) — 체력 비율이 가장 낮은 아군에게 보호막 10 부여

**법사 (Mage)**
- **전격** (AP 15) — 타겟과 인접한 적에게 피해 + 감전(이동 불가)
- **텔레포트** (AP 20) — 체력 최저 아군을 법사 옆으로 소환 (시전자 탈진)
- **파괴광선** (AP 30) — 직선 방향 모든 적에게 대규모 피해

## 최종 보스: 드래곤 패턴
- **기본 정보**: HP 200 / 공격력 10 / 방어력 3, 이동하지 않으며 맵 중앙 **7×7 영역**(좌표 4~10, 4~10)을 차지합니다. 모든 상태 이상에 면역입니다.
- **역린(약점) 시스템 ⭐**: 몸통 한쪽에 약점 타일이 **`R`** 로 표시됩니다. `R`을 공격하면 **방어력 무시 + 2배 피해**가 들어가고, 일반 `D` 부위는 (피해 − 방어력 3)만 적용됩니다. 약점 위치는 랜덤이며 **3턴마다 다른 방향으로 이동**하므로, 매번 바뀌는 `R`을 노리는 것이 공략의 핵심입니다.
- **행동 규칙**: 좌·우 가장자리(x=3 또는 x=11)에 아군이 붙어 있으면 **날갯짓**을 우선 사용하고, 그렇지 않으면 아래 스킬 중 **직전에 쓴 스킬을 제외하고 무작위**로 하나를 사용합니다.

| 스킬 | 효과 |
|------|------|
| 메테오 | 무작위 아군 1명 중심 5×5 범위에 10 피해 |
| 드래곤의 공포 | 모든 아군을 1턴 기절시킴 |
| 지진 | x좌표 4~10에 있는 모든 아군에게 8 피해 |
| 날갯짓 | 좌·우 가장자리(x=3/11)의 아군을 2칸 밀어내고 5 피해 + 기절 |
| 드래곤 브레스 | 타워(기지)에 직접 10 피해 |

> **공략 팁**: 좌·우 끝(x=3/11)에 붙으면 날갯짓에 밀려나며 기절하니 위·아래쪽에서 접근하고, 메테오·지진을 대비해 유닛을 분산 배치하세요. 매 턴 바뀌는 `R` 역린을 집중 공격하는 것이 가장 빠른 처치법입니다.

## 보물상자
- **등장**: 웨이브마다 빈 칸 한 곳에 보물상자(`$`, 주황 타일)가 1개 등장합니다. (적 스폰행·기지 근처는 제외)
- **개봉**: 유닛을 상자에 **인접**시킨 뒤 `e` 키를 누르면 열립니다. 개봉에는 **AP 20**이 소비됩니다. (상자에 인접하면 안내 문구가 표시됩니다)
- **버프(무작위, 영구 지속)**: 상자를 연 캐릭터에게 아래 중 하나가 무작위로 적용되며, 한 번 얻으면 계속 유지됩니다.

| 버프 | 효과 |
|------|------|
| 최대 체력 증가 | 최대 체력 +20, 현재 체력 +20 (기존 한계를 넘어 회복) |
| 공격력 증가 | 공격력 +5 (영구) |
| 최대 AP 증가 | 최대 AP +20, 현재 AP +20 (영구) |
| 방어력 증가 | 방어력 +3 (받는 피해 감소, 영구) |

> 획득한 버프는 세이브/로드 후에도 유지됩니다.

## 주요 기능
맵은 15 × 15 격자로 구성되며, 생성 시 타워(기지)를 배치하고 산·강 등의 지형을 초기화합니다.
```cpp
// Map.cpp
GameMap::GameMap(int w, int h) : width(w), height(h) {
    grid.assign(height, std::vector<TileType>(width, TileType::EMPTY));
    grid[height - 1][width / 2] = TileType::BASE;  // 하단 중앙에 타워 배치
    initTerrain();                                 // 산·강 지형 초기화
}
```

웨이브가 넘어갈 때마다 아군 위치와 기지를 보호하면서 지형을 새로 생성하고, BFS로 적의 진입 경로가 막히지 않도록 검증합니다.

각 유닛은 `Character`를 상속하며, 스킬은 `Skill`을 상속해 다형적으로 실행됩니다.
```cpp
// Skill 다형성 예시 - 회전베기(SpinSlash)
bool execute(Character& caster, GameState& state) override {
    for (auto e : state.getEnemies()) {
        if (!e->isAlive()) continue;
        // 기사 주변(대각선 포함, 체비셰프 거리) 이동 범위 내 적이면 피해
        // ...
        e->takeSkillDamage(caster.atk, e->x, e->y);
        state.addHitEffect(e->x, e->y, caster.atk);
    }
}
```
